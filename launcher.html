You can play Minecraft on PC/Mac/Browser. It can be played in the browser with Eaglercraft, an AOT compiled voxel game inspired from Minecraft designed to run on Javascript. It uses TeaVM and LAX1DUDE's OpenGL emulator to run a Java virtual machine fully compatible with browsers. Minecraft: Pocket Edition is available for iOS and Android, and Minecraft: Xbox 360 Edition is available on the XBLA Marketplace.
